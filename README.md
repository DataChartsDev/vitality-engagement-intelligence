# Vitality Engagement Intelligence Engine

An end-to-end machine-learning product for predicting near-term wellness-programme disengagement, selecting an appropriate behavioural intervention, and generating a safe personalised message.

## Current status

Stage 2 complete: synthetic dataset generation, validation, export, inspection, and documentation.

## Core technologies

- Python
- SQL
- BigQuery
- Vertex AI
- FastAPI
- Docker
- GitHub Actions

## Important limitation

This project uses synthetic wellness-engagement data. Results generated from synthetic data do not demonstrate real-world health or behavioural effectiveness.
